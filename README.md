## How to use it

**Instalar TESTRPC**

```
npm install -g ethereumjs-testrpc
```

**Inicializar TESTRPC**

```
testrpc --l 500000000
```

**Instalar modulos con npm**
```
npm install
```

**Compilar**
```
truffle compile
```
**Migracion de contratos**
```
truffle migrate
```

**Test de contratos**
```
truffle test
```
**Run dev server**
```
npm run start
```
**Build for production**
```
npm run build
```

La aplicación se inicia por defecto en localhost:3000
